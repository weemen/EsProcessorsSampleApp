# Sample App for using ES-Processor library.
This app demonstrates how to use the [ES-Processor](https://github.com/weemen/es-processors) library works.

In main.scala you can see all the code that is needed to create a processor to use in your eventsourcing project.

## Run it
To run this sample app, follow these steps:
1. Clone the repository.
2. Install dependencies using sbt.
3. Run the application with `sbt run`.
